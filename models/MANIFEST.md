# Model Manifest

No model weights are bundled with this repository.

## Policy

- Keep model weights out of git.
- Review each model license before installation.
- Do not assume commercial rights from an open-source code license.
- Store local model paths in machine-local config only.
- Never commit provider keys, model access tokens, or vendor credentials.

## Candidate Engines

| Engine | Use | Notes |
| --- | --- | --- |
| FaceFusion | Local face-swap bootstrap | Review OpenRAIL-AS and bundled model licenses before commercial use. |
| Pixverse/SWAP API | Cloud video swap | Use only if API terms and commercial rights are acceptable. |
| HeyGen | Talking-avatar UGC | Good for presenter ads and scripted creator clips. |
| Replicate | Hosted model experiments | Review each model card/license individually. |
